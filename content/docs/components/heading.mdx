## Usage

All headings in MDX automatically get anchor links. This allows users to link directly to specific sections of a page.

### Example

```mdx
## My Section

Content under this section.

### Subsection

More content here.
```

Headings generate IDs based on their text content. For example, `## My Section` generates the ID `my-section`, which can be linked to via `#my-section`.

## Heading Levels

# Heading 1
## Heading 2
### Heading 3
#### Heading 4
##### Heading 5
###### Heading 6

The table of contents on the right sidebar is automatically generated from h2 and h3 headings.
