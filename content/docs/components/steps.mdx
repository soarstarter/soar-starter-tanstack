## Usage

<Steps>
  <Step>
    ### Get Starter Template

    ```bash
    pnpm create next-app my-docs
    ```
  </Step>
  <Step>
    ### Install Dependencies

    ```bash
    pnpm add fumadocs-core fumadocs-ui fumadocs-mdx
    ```
  </Step>
  <Step>
    ### Development Server

    ```bash
    pnpm dev
    ```
  </Step>
</Steps>
