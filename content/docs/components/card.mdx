## Usage

<Card title="Card Title" description="Card description with details." />

```mdx
import { Card, Cards } from 'fumadocs-ui/components/card';

<Card title="Card Title" description="Card description with details." />
```

### With Link

<Card
  title="Link Card"
  description="Beautifully designed documentation template. Customizable. Compatible. Open Source."
  href="https://github.com"
/>

```mdx
<Card
  title="Link Card"
  description="Click to navigate."
  href="https://github.com"
/>
```

## Props

### Card

| Prop        | Type     | Default |
| ----------- | -------- | ------- |
| title       | `string` | -       |
| description | `string` | -       |
| href        | `string` | -       |
| icon        | `ReactNode` | -    |
| external    | `boolean` | -      |

### Cards

| Prop | Type     | Default |
| ---- | -------- | ------- |
| children | `ReactNode` | -  |
